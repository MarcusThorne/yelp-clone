Setting up a PERN Stack

1. Create a folder MKDIR app name and then create a server MKDIR server, then a client npc create-react-app client.

2. Cd into the server and run npm init -y, followed by npm install express dotenv nodemon
2a. Add to following line after "test" in json file
    "test":  "echo \"Error: no test specified\" && exit 1",
    "start": "nodemon index.js" <-- add this one

3. Create a index.js file and add the following code
  // Requring the dotenv file
  require('dotenv').config();

  // Setting up express and calling it
  const express = require('express');
  const app = express();

  // Requiring loaders for the app from an index file in the loaders directory
  const loaders = require('./loaders');

  // Defining the port based on the env file
  const port = process.env.PORT || 4000;

  async function startServer() {

    // Init application loaders
    loaders(app);

    // Start server
    app.listen(port, () => {
      console.log(`Server listening on port ${port}`);
    })
  }

  startServer();


4. Create a .env file in server directory
4a. Add a PORT variable to env file and define 4000 or whatever number you want
4b. Add require('dotenv').config(); to top of index.js file

5. create a loaders folder with an index.js

6. create a routers folder with an index.js
